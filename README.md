<h2>How to</h2>
<pre><code>
Add it in your root build.gradle at the end of repositories:

	allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
    
Add the dependency

	dependencies {
	        compile 'com.github.duyangs:ZBaseLib:0.1.4'
	}
    
</code></pre>
